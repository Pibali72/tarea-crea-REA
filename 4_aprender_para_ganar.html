Lumen dice: ¿Sabes utilizar ya las diferentes medidas de longitud?

Las medidas de longitud que ya conoces son:

  • El Km, se llama kilómetro y es el mayor de todos, lo usamos para distancias muy largas.
  • El m, se llama metro y solemos usarlo para distancias cercanas.
  • El cm, se llama centímetro y con el se suele medir todo lo que cogemos con nuestras manos,.
  • El mm, se llamo milímetro, y cuando algo es muy pequeño es la unidad de medida que utilizamos, como un grano de arroz.
